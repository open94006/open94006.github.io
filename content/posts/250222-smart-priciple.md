### 可以將 SMART 單字拆分成以下五種含義

-   明確（Specific）
-   可衡量（Measurable）
-   可實現（Achievable）
-   相關（Relevant）
-   有時間限制（Time-bound）

```
以學習「系統分析與設計」為例
```

| 項目       | 描述                                                 |
| ---------- | ---------------------------------------------------- |
| Specific   | 有設計才有開發的規範，重點是讓我不會一直抱怨         |
| Measurable | 可初步簡單設計一個電商的結帳流程與建立訂單           |
| Attainable | （待定）需要有個軟體可線上開發拓展，並持續維護與改良 |
| Relevant   | 與專案的開發有高度關聯                               |
| Time-based |                                                      |
